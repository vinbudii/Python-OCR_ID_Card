# Python-OCR_ID_Card

## Installation

-   Pull code from repo
    ```
    git pull https://github.com/vinbudii/Python-OCR_ID_Card.git
    ```
-   Install Python
    ```
    https://www.python.org/downloads/
    ```
-   Install OpenCV
    ```
    pip install opencv-python
    ```
-   Install tkinter
    ```
    pip install tk
    ```
-   Install easyocr
    ```
    pip install easyocr
    ```
-   Install pillow
    ```
    pip install Pillow
    ```
